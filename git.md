# Git

Git - это система контроля версий. Она помогает сохранять историю изменений в проекте и загружать файлы на GitHub.

## Основные понятия

**Репозиторий** - папка проекта, за которой следит Git.

**Commit** - сохраненная версия изменений. Это как контрольная точка.

**GitHub** - сайт, где можно хранить репозитории онлайн.

**Push** - отправить изменения с компьютера на GitHub.

**Pull** - скачать последние изменения с GitHub на компьютер.

## Как обычно работать

Перед началом работы подтянуть свежие изменения:

```bash
git pull
```
