# notes

Личный репозиторий для заметок, шпаргалок и полезных команд.

## Что здесь лежит

- `git.md` - заметка по основным командам Git
- позже можно добавить заметки по Linux, Docker, DevOps, Markdown и другим темам

## Как я веду заметки

Заметки пишутся в формате Markdown (`.md`).

Пример структуры:

```text
notes/
  README.md
  git.md
  linux.md
  docker.md
  devops/
    ci-cd.md
    nginx.md
```

## Быстрый цикл работы

Перед началом работы:

```bash
git pull
```

После редактирования заметок:

```bash
git status
git add .
git commit -m "обновил заметки"
git push
```

## Правила для себя

- писать коротко и понятно
- одна тема - один файл
- важные команды добавлять с примерами
- после изменений не забывать делать `git push`
