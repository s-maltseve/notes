# 📚 NOTEBOOK 📚

Личный репозиторий для заметок, шпаргалок, учебных материалов и полезных команд.

## Что здесь лежит

- `NOTEBOOK/TOPICS/git.md` - заметка по основным командам Git
- `NOTEBOOK/TOPICS/markdown.md` - шпаргалка по форматированию Markdown
- `NOTEBOOK/MATERIALS/source.md` - источники, roadmap и полезные ссылки
- `NOTEBOOK/PROJECTS/` - папка для будущих проектов и практических задач
- `NOTEBOOK/ASSETS/` - картинки, схемы и другие файлы для заметок

## Как я веду заметки

Заметки пишутся в формате Markdown (`.md`).

Одна тема - один файл.  
Картинки и схемы хранятся отдельно в `ASSETS`.

Пример структуры:

```text
notes/
  README.md
  NOTEBOOK/
    PROJECTS/
    TOPICS/
      git.md
      markdown.md
    MATERIALS/
      source.md
    ASSETS/
      roadmap_devops.jpg
