# Markdown: большая шпаргалка по форматированию текста

Markdown - это простой формат для заметок, README, документации и шпаргалок.

## 1. Заголовки

Заголовки делаются через `#`.

```markdown
# Заголовок 1 уровня
## Заголовок 2 уровня
### Заголовок 3 уровня
#### Заголовок 4 уровня
##### Заголовок 5 уровня
###### Заголовок 6 уровня
```

Чаще всего хватает:

```markdown
# Название заметки

## Большой раздел

### Подраздел
```

---

## 2. Обычный текст

Обычный текст пишется без специальных знаков.

```markdown
Это обычный текст.

Это новый абзац.
```

Чтобы сделать новый абзац, оставь пустую строку между строками.

---

## 3. Перенос строки

Если просто нажать Enter, Markdown может склеить строки.

Варианты:

```markdown
Первая строка.  
Вторая строка.
```

Или лучше через пустую строку:

```markdown
Первый абзац.

Второй абзац.
```

---

## 4. Жирный текст

```markdown

```

Пример:

**жирный текст**

## 5. Курсив

```markdown

```

Пример:

## 6. Жирный курсив

```markdown

```

Пример:

---

## 7. Зачеркнутый текст

```markdown
~~зачеркнутый текст~~
```

Пример:

~~зачеркнутый текст~~

---

## 8. Код в строке

Используется для команд, файлов, переменных и коротких фрагментов кода.

```markdown
Команда `git status` показывает состояние репозитория.
```

Пример:

Команда `git status` показывает состояние репозитория.

---

## 9. Блок кода

Блок кода делается тремя обратными кавычками.

````markdown
```bash
git status
git add .
git commit -m "обновил заметки"
git push
```
